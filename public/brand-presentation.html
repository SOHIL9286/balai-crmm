File Formats

Professional format guide for different use cases

Format Use For Recommended Size Notes
SVG Vector Graphics Infinite Scale Best for websites. Scalable without quality loss. Master format.
PNG Transparent Images 1024, 512, 256, 128, 64 Best quality. Use for high-fidelity needs. Export at 2x resolution.
WEBP Web Optimized Same as PNG Fast loading. Modern browsers. 25-35% smaller than PNG.
JPG Documents & Print 180 × 50 px No transparency. Good for traditional print and documents.
ICO Favicon 16×16, 32×32, 48×48 Browser icon. Multiple sizes in one file. Use favicon.ico

Logo Usage Guidelines

Best practices for maintaining brand consistency

✓ Do's
  • Maintain clear space (minimum 20% of logo height)
  • Keep original brand colors vibrant
  • Use SVG format whenever possible
  • Preserve original aspect ratio
  • Use transparent PNG when needed
  • Keep minimum size at 120px width
  • Test on all intended backgrounds
✕ Don'ts
  • Never stretch or distort the logo
  • Never rotate or skew the logo
  • Never add shadows or effects
  • Never change brand colors
  • Never place on busy backgrounds
  • Never use blurred or low-resolution versions
  • Never combine with other logos
🎨 Brand Asset Management
Keep the original SVG file as the master logo.
Export PNG, WEBP, JPG and ICO formats from the SVG master file.
Store all formats in organized folders: /logos/{format}/{size}/
Update this presentation whenever logo variants are modified.
