## Notes

- Designed for PostgreSQL 16 and Supabase compatibility.
- Includes multi-tenant organization and branch support.
- Includes seed data for customers, vehicles, policies, quotes, claims, renewals, leads, follow-ups, payments, finance, RTO services, users, insurance companies and activity logs.
- Includes storage bucket recommendations, RLS policies and backup guidance.
